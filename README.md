# imapfilter-spamfilter-config

A simple spamfilter configuration using imapfilter.

Train your servers central spamfilter using information from Thunderbird or AppleMail, where the user marked e-mails as "Spam" or "Ham".

Then remotely spam-filter mails in your IMAP Inbox using that information.

requires that you disable Spam-Filtering in Thunderbird/AppleMail and only manually mark stuff as Spam or Ham.
